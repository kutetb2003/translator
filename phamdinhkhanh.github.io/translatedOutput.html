2.9.Chain rule in the opposite spread (Backpropagation)¶

The Backpropagation process will perform the gradient descent calculation and update the coefficient.Backpropagation can be considered as _Linh soul _ of the training process of neural networks because there is no them, we cannot update the neural network coefficient (weights) to advance to local extreme points.The Backpropagation formula will do two things:

  • Calculate the gradient descent at each layer from the end to the beginning.So this formula is called the reverse spread.

  • Update gradient descent at each layer according to the value above.

To calculate the gradient descent in the BackPropagation, we rely on the chain rule formula.For example, in the above diagram, the nodes and blue arrows will go in the opposite direction with the black arrow, representing the gradient descent according to the chain rule.

Suppose we consider the backpropagation process at a specific node with inputszAnd the output isH.H=f(z).

In Backpropagation, from outputS, to calculate the derivativeΔSΔzWe will not count directly but must go through the component derivatives in the chain rule as the following derivativeHAnd then followz:

ΔSΔz=ΔSΔHΔHΔz

The gradient descent is done at the node calledLocal Gradient, Gradient Descent at the back of the node isDownstream GradientAnd the front node isupSream Gradient.The above formula can be encapsulated into:

downstream=upSream×Local
